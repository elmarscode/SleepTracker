I used an Android app called Accelogger to record the raw data from the phone accelerometer. 
The format is 
“Timestamp, number, x-axis, y-axis, z-axis”.
Sample: 
“2015/03/05 23:24:18,137131418001491,2.030670,4.797090,8.534699”


I am using Octave with a GUI to write and test my code. (https://www.gnu.org/software/octave/download.html)

I am using the MXE build (http://mxeoctave.osuv.de/ it is unofficial, but recommended by the wiki http://wiki.octave.org/Octave_for_Windows)
